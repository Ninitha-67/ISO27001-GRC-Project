# ISO/IEC 27001 GRC Project

An Excel-based governance, risk, and compliance workbook for ISO/IEC 27001 risk tracking and control mapping.

## Project Overview
This project demonstrates an information security management system workflow using Excel.

It includes:
- Risk Register
- Risk Assessment with likelihood x impact scoring
- Risk Matrix
- Annex A control mapping
- Statement of Applicability
- Compliance and evidence tracking
- Dashboard reporting

## Features
- Risk Register with likelihood x impact scoring
- ISO/IEC 27001:2022 Annex A control reference set (93 controls)
- Statement of Applicability workbook
- Control-risk mapping
- Risk trend tracking
- Dashboard with KPI summaries and charts
- Compliance Register
- Audit Evidence tracking

## Quick Start
1. Open `ISO27001_GRC_Project_v4.xlsx`
2. Review the `Risk Register` sheet
3. Enter your organization risks
4. Use the dropdowns for consistent values
5. Review `Dashboard` for status and trends

## Risk Calculation Model
Risk Score = Likelihood x Impact
