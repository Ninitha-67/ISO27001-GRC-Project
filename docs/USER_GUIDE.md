# User Guide

## Purpose
This workbook helps track risks, map them to controls, and monitor remediation progress.

## Workbook Sheets
- `Dashboard`: summary KPIs and charts
- `Risk Register`: core risk list
- `Risk Matrix`: scoring reference
- `SoA`: applicability and implementation tracking
- `Annex A Controls`: ISO/IEC 27001:2022 control reference list
- `Control-Risk Mapping`: control linkage view
- `Risk Trends`: monthly tracking
- `Lists`: dropdown values

## How to Use
1. Add or update risks in `Risk Register`.
2. Select likelihood, impact, owner, and status from dropdowns.
3. Review the calculated risk score and level.
4. Update `SoA` for control applicability and implementation.
5. Check `Dashboard` for overall progress.

## Color Legend
- Red: Critical
- Orange: High
- Yellow: Medium
- Green: Low

## Tips
- Keep risk names short and specific.
- Use one owner per risk.
- Update statuses regularly.
- Review controls after major changes.
